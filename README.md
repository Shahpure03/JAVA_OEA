# 📘 Quiz Management System (Java)

## 📌 Project Idea
The **Quiz Management System** is a console-based Java application designed to simulate a real-world quiz platform. It allows faculty to create quizzes and students to attempt them, with automatic evaluation and result generation.

The system manages students, quizzes, questions, and scores efficiently using Java programming concepts and data structures.

---

## 🛠️ Features
- 👩‍🎓 Student Registration  
- 📝 Quiz Creation and Management  
- ❓ Multiple Choice Questions (MCQs)  
- ✍️ Quiz Attempt by Students  
- ✅ Automatic Answer Evaluation  
- 📊 Score Calculation and Result Storage  
- 📈 Leaderboard Generation (Ranking System)  
- 🔍 Filtering and Searching using Stream API
